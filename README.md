# 🚀 Atlanta Software

Este é o site pessoal da empresa Atlanta, onde são apresentados os softwares desenvolvidos pela mesmo, como gestão docente, bancária e sistema médico. O site foi construído com foco em ser leve, funcional e de fácil hospedagem gratuita.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3** - Estrutura e estilização (com componentes Bootstrap).
* **JavaScript (ES6+)** - Lógica de interação e integração de APIs.
* **[Formspree](https://formspree.io/)** - Serviço de recebimento de formulários via AJAX.
* **WhatsApp API** - Integração de botão flutuante para contacto direto.
* **GitHub Pages** - Hospedagem estável e gratuita.
